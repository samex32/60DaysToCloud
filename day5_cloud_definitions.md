# Cloud Service Models

## IaaS (Infrastructure as a Service)
- **Definition:** Renting the raw hardware/virtual machines.
- **Example:** AWS EC2
- **Who uses it:** System Admins, DevOps Engineers.

## PaaS (Platform as a Service)
- **Definition:** Renting a platform to build and run apps without managing servers.
- **Example:** Heroku, Google App Engine.
- **Who uses it:** Developers who just want to deploy code.

## SaaS (Software as a Service)
- **Definition:** Finished software accessed via internet.
- **Example:** Gmail, Zoom.
- **Who uses it:** End Users.
